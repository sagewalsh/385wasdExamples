# WASD Basic Movements

I | Zoom the camera in
O | Zoom the camera out

W | Move box up
S | Move box down
D | Move box right
A | Move box left

E | Move Diagonally right, up
Q | Move Diagonally left, up
X | Move Diagonally right, down
Z | Move Diagonally left, down

P | Pause the game (again to resume)

esc | Quit the game
